Utilizando esta base del nuestro mini-CRM (Content Management System):

Clónalo a tu servidor local y crea una web personalizada
Respeta la estructura principal: a. data.json para los datos que se repitan - site: para los datos generales del sitio (titulo, descripcion, telefono, correo, redes sociales, etc..) - array: para listas de valores con los que interactural (productos, alumnos, personajes, peliculas, servicios, lugares, videos, galería, animales, plantas, planetas, etc..) b. config.php para constantes que tengan que ver con la configuración del site/app (en esta fase casi no haría falta tocar nada selvo DEBUG) c. views/ para los apartados o vistas (home.php, servicos.php, contacto.php, etc...) d. style.css para la apariencia de todo el site/app
